Mimivoice is a personal project inspired by a simple question: **what if voice training felt less intimidating and more like learning an instrument?**

Originally created to help transgender women—including myself—practice vocal feminization, the project aims to transform repetitive exercises into a calm, motivating experience.

Rather than replacing traditional coaching, Mimivoice focuses on making consistent practice easier through immediate visual feedback and thoughtful interface design.

## Vision

The long-term goal is to create a training environment that feels supportive instead of clinical.

Planned features include:

- Live microphone analysis.
- Pitch and resonance visualization.
- Guided practice sessions.
- Progress tracking over time.
- Personalized exercises.
- A clean, distraction-free interface.

## Tech

- React
- TypeScript
- Web Audio API

Mimivoice is currently in active development.
